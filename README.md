# Virtual_hand_Mouse_Project
An AI-based Virtual Mouse built using Computer Vision that allows users to control the cursor, perform clicks, scrolling, and screenshots using real-time hand gestures.

🖐️ AI Virtual Hand Mouse

This project is a Computer Vision-based Virtual Mouse that allows users to control their system cursor using real-time hand gestures captured through a webcam.

Built using:

Python

OpenCV

MediaPipe

PyAutoGUI

Pynput

The system tracks hand landmarks and interprets specific finger gestures to perform mouse operations without physical contact.
🚀 Features

✔ Cursor Movement using Index Finger
✔ Left Click
✔ Right Click
✔ Double Click
✔ Scroll Up & Down
✔ Screenshot Capture
✔ Real-time Hand Tracking
✔ Single-Hand Detection
🧠 How It Works

1.MediaPipe detects 21 hand landmarks.

2.Finger angles and distances are calculated.

3.Gestures are mapped to specific mouse actions.

4.PyAutoGUI and Pynput simulate mouse operations.

🎯 Applications

-Touchless human-computer interaction

-Accessibility support

-Smart systems & automation

-Gesture-based AI interfaces
